checkproxy
==========

An application that checks if a proxy can be used to access the internet.

Useful for students in colleges where internet access is restricted through proxy servers and not all of them are working 24/7.

## Usage

Requires [Shoes](http://shoesrb.com) to be installed. Downloads can be found [here](http://shoesrb.com/downloads/).

#### Instructions

In **Shoes**, choose **Open an App** and select the `checkproxy.rb` file.

###### OR

Run as `path_to_shoes_installation/shoes checkproxy.rb`
